# GenZ-MicroLuxury-Pivot-Analysis
Econometric analysis of Indian Gen Z consumption patterns (2024-2026) featuring the Luxury Displacement Index (LDI)

Research Title: The Micro-Luxury Pivot Summary: Investigating why Indian Gen Z reallocates capital to "soft assets" once the 50% EMI threshold is breached. Technical Stack: Python, Random Forest Regression, LDI & FSI Indices. Data Sources: 6 Kaggle Datasets (Pande, Halper, Kumar, etc.)
